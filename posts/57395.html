Docker总结

发表于2024-01-03|更新于2024-01-05|Docker
|阅读量:|评论数:

配置虚拟机固定ip

vi /etc/sysconfig/network-scripts/ifcfg-ens33

添加配置

IPADDR=”192.168.200.128”//IP
NETMASK=”255.255.255.0”//子网掩码
GATEWAY=”192.168.200.2”//网关

开启mysql的权限

grant all on . to root@’%’ identified by ‘Lxx@123’ with grant option;
刷新权限flush privileges;

查询是否自启动

systemctl list-unit-files | grep docker

设置自启动

systemctl enable docker.service

设置容器自启动

docker update –restart=always 容器名称

在配置mysql3306端口时没效果时执行如下

docker run –name mysql -p 3306:3306 -e MYSQL_ROOT_PASSWORD=Lxx@123 -d mysql:latest

在配置redis6379端口时没效果时执行如下

docker run -p 6379:6379 –name myredis -v /home/redis/redis.conf:/etc/redis/redis.conf -v /home/redis/data:/data -d redis redis-server /etc/redis/redis.conf –requirepass “Lxx@123”

若shell连接虚拟机失败

vi sshd_config
修改#usDNS=YES 为usDNS=no
再执行
systemctl restart sshd.service

进入docker容器中
docker exec -it mysql /bin/bash

若是电脑断电打不开可执行

xfs_repair -v -L /dev/dm-0

//查看镜像目录

docker ps

//查看文件

docker images

//停止镜像

rm stop xxx

//删除镜像

rmi -f xxx

//创建文件

mkdir xxx(目录)

//删除文件

rm -f xxx(文件)

//删除文件夹

rm -rf xxx(文件夹)

前端发包流程

1.上传压缩包到指定目录
2.unzip xxx(压缩包)
3.rm -f xxx(压缩包)
